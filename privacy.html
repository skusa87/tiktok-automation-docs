Privacy Policy

Last updated: July 18, 2025

1. Application Overview

This privacy policy describes how our TikTok automation testing application handles information when you use our service for educational and testing purposes.

2. Information We Access

Our application may access the following information through TikTok API:

  • Basic user profile information (username, user ID)
  • Video upload capabilities to your TikTok account
  • Video metadata for uploaded content

3. How We Use Information

The accessed information is used solely for:

  • Uploading educational quiz videos to your TikTok account
  • Testing API integration functionality
  • Educational purposes and learning TikTok API development

4. Data Storage and Retention

We do not store personal information permanently. Any temporary data used during the upload process is:

  • Processed locally on your system
  • Not shared with third parties
  • Deleted after processing completion

5. Data Security

We implement appropriate technical measures to protect your information during the API interaction process. All communications with TikTok API use secure HTTPS connections.

6. Third Party Services

This application interacts with:

  • TikTok API (governed by TikTok's Privacy Policy)
  • Local file system for video processing
  • Google Sheets for workflow tracking (if used)

7. Your Rights

You have the right to:

  • Revoke API access at any time through TikTok settings
  • Request information about data processing
  • Contact us with privacy concerns

8. Testing and Development

This application is in testing phase. Data handling practices may evolve as development progresses. Users will be notified of any significant changes to privacy practices.

9. Contact Information

For privacy-related questions or concerns, please contact the developer through GitHub.

10. Changes to Privacy Policy

We may update this privacy policy as the application develops. The "Last updated" date at the top of this policy indicates when it was last revised.
